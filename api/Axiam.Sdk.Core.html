Namespace Axiam.Sdk.Core

Classes

AuthError

Authentication failure: wrong credentials, expired session, MFA failure, or a 401 on refresh (CONTRACT.md §2, D-12). Always constructed via ErrorMapper so REST and gRPC transports cannot drift on the error taxonomy.

AuthzError

Authorization failure: the caller lacks permission for the requested operation (CONTRACT.md §2, D-12). Always constructed via ErrorMapper so REST and gRPC transports cannot drift on the error taxonomy.

ErrorMapper

Central status→error mapper (CONTRACT.md §2, D-12). The single source of truth for both the HTTP status table and the gRPC status-code table so the REST and gRPC transports cannot drift on the error taxonomy.

NetworkError

Transport-level failure: connection refused, timeout, TLS error, DNS failure, or a server-side 5xx (CONTRACT.md §2).

SensitiveJsonConverterFactory

Factory that produces the closed SensitiveJsonConverter<T> for a given Sensitive<T>. An open-generic converter referenced from a JsonConverterAttribute MUST be a JsonConverterFactory: System.Text.Json cannot instantiate an open-generic JsonConverter<T> directly (it throws because the type still contains generic parameters).

SensitiveJsonConverter<T>

System.Text.Json converter for Sensitive<T>. Write always emits the redacted literal; Read is intentionally unsupported — a Sensitive<T> is a write-only-for-serialization type, so no wire format can ever deserialize a real value back into one.

TenantContext

Immutable tenant-identity value object required at AxiamClient construction time and propagated on every outgoing request (CONTRACT.md §5: X-Tenant-Id header on REST; x-tenant-id gRPC metadata in a later plan). There is no default-constructible/blank instance — the constructor throws a runtime guard backing the compile-time guarantee that AxiamClient's own tenant-required constructor provides (SC#1): AXIAM is multi-tenant and there is no default tenant.

Structs

Sensitive<T>

Wraps a token-carrying value so it can never be accidentally exposed via ToString(), System.Text.Json serialization, or a public getter (CONTRACT.md §7, D-12). Only SDK-internal code may construct an instance or read the wrapped value, via Axiam.Sdk.Core.Sensitive<T>.Reveal() — there is intentionally no public accessor.
