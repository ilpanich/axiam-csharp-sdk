MinKeyVersion

The lowest AMQP signed-envelope key_version this SDK will accept. TryExtractMetadata(byte[], out ReplayMetadata)

Parses key_version, nonce, and issued_at out of an already-verified message body. Never throws: a malformed body, or one missing nonce/issued_at entirely, returns false (which Check(ReplayMetadata)'s caller must treat as a rejection — an unparseable/absent replay-protection field is exactly as unsafe as a stale one). A missing/non-numeric key_version yields KeyVersion == 0, which is always rejected by Check(ReplayMetadata) since it is below MinKeyVersion.
