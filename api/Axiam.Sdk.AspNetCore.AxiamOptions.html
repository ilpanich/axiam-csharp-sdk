Configured via services.AddAxiamAspNetCore(options => { ... }) — properties are mutable (set, not init) because the .NET Options pattern configures an already-constructed instance through an Action<T> delegate.

DefaultTenantId

The tenant this ASP.NET Core app instance serves. Used for two purposes: (1) to construct the single shared AxiamClient that AddAxiam registers, and (2) as AxiamAuthMiddleware's fallback tenant when an incoming request does not carry an X-Tenant-ID header (CONTRACT.md §5/§10: "Reads X-Tenant-ID (or configured tenant)"). Required — there is no silent default tenant anywhere in this SDK; a missing value surfaces as a startup-time ArgumentException from AxiamClient's own tenant-required constructor (SC#1) the first time the DI container resolves it.

public required string DefaultTenantId { get; set; }
