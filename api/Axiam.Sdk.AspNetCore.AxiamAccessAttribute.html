Declarative, per-endpoint AXIAM authorization attribute (CONTRACT.md §11 require_access(action, resource[, scope]) — C# maps this to [AxiamAccess(...)]). Sugar over the existing "resource:action" policy string mechanism (AxiamPolicyProvider/AxiamPolicyHandler): this attribute serializes its Action/Resource/ Scope/ResourceRouteParam into a structured policy name ("axiam::<action>::<resource>::<scope>::<param>") that AxiamPolicyProvider recognizes and parses ALONGSIDE the legacy single-colon "resource:action" form (which remains fully supported — see its own remarks). Because this type derives from AuthorizeAttribute, it can be placed directly on a controller/action ([AxiamAccess("read", "documents")]) or passed to RequireAuthorization(IAuthorizeData) for minimal APIs — ASP.NET Core's own authorization pipeline combines it into a policy via Policy exactly like any other AuthorizeAttribute.

Remarks

This attribute never calls the AXIAM server itself — it only computes a policy name. The actual check happens in AxiamPolicyHandler, strictly AFTER the §10 authentication middleware (AxiamAuthMiddleware) has already verified the caller and populated User; this attribute never performs its own token extraction or verification (CONTRACT.md §11.2.1).

Constructors

AxiamAccessAttribute(string, string?)

Constructs the attribute from the mandatory action and the optional resource half (CONTRACT.md §11.1: argument order is always action before resource). Scope and ResourceRouteParam may be set afterwards via object-initializer syntax (e.g. [AxiamAccess("read", "documents")] or new AxiamAccessAttribute("read", "documents") { Scope = "team-a" }); every such assignment recomputes the underlying policy name.

ResourceRouteParam

The name of the route/path value whose UUID value identifies the target resource (CONTRACT.md §11.2.3b). Defaults to DefaultResourceRouteParam ("id"). A missing or non-UUID route value at request time is a 400 invalid_request outcome (§11.2.3) — never a silent allow, never a nil/empty-UUID fallback. Setting this property recomputes Policy.
