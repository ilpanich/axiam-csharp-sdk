FromHttpStatus(HttpStatusCode, string)

Overload for callers that only have the status code (no live response) — e.g. a pre-parsed error body. Never accepts a raw response; use FromHttpResponse(HttpResponseMessage, string) when a live HttpResponseMessage is available so NetworkError can build its redacted header summary.
