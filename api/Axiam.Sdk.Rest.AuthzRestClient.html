Remarks

This class holds NO local cache of any authorization decision — every call hits the server fresh. AXIAM's RBAC engine is additive-only (allow-wins, default-deny, SEC-040; project constraint per CLAUDE.md); a client-side cache or short-circuit could silently diverge from the server's live decision, which this SDK must never risk.
