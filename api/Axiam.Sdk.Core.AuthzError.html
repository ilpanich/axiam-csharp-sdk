AuthzError(string, string?, string?)

Constructs an AuthzError carrying the structured action/ resource_id fields parsed from the server's 403 authorization_denied body by ErrorMapper. Either may be null — action when not known, resourceId when the denial was not resource-scoped.

Properties

Action

The denied action (e.g. "users:get"), when known. Populated from the server's structured 403 body's action field (CONTRACT.md §2: AuthzError SHOULD carry the denied action/resource_id if available). Null when the transport has no body to parse (gRPC PERMISSION_DENIED) or the field was absent.

public string? Action { get; }

Property Value

string

ResourceId

The resource UUID the denial was scoped to. Populated from the server's structured 403 body's resource_id field — present only for a resource-scoped denial. Null for a non-resource-scoped denial or when the transport has no body to parse (gRPC PERMISSION_DENIED).
